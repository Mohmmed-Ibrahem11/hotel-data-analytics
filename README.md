# Hotel Data Engineering & Analytics Project 🏨

## 📌 Project Overview
An end-to-end data solution for hotel management, bridging the gap between business requirements and technical data infrastructure.

## 🛠️ Tech Stack & Workflow
1. **Business Analysis**: Analyzed hotel business requirements, identified key pain points, and defined critical business KPIs (Revenue, Occupancy, and Cancellations).
2. **Database Design (Draw.io)**: Created the Entity-Relationship Diagram (ERD) based on the business analysis to structure hotel operations efficiently.
3. **Data Infrastructure (SQL)**: Built the database schema, loaded data, and created a dedicated `Date Dimension` table for advanced analytics.
4. **Business Intelligence (Power BI)**: Developed an interactive executive dashboard to visualize the insights and help hotel managers make data-driven decisions.
## ⚙️ Environment & Tools
* **Database Engine**: Microsoft SQL Server (or your specific SQL database)
* **Design Tool**: Draw.io (Browser Version)
