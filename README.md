# Hotel Data Engineering & Analytics Project 🏨

## 📌 Project Overview
An end-to-end data solution for hotel management, covering everything from database creation to interactive business intelligence reporting.

## 🛠️ Tech Stack & Workflow
1. **Data Infrastructure (SQL)**: Built the database schema, loaded data, and created a dedicated `Date Dimension` table for advanced time-intelligence analytics.
2. **Analytical Queries (SQL)**: Wrote advanced queries to extract business metrics like Revenue, Occupancy Rates, and Booking trends.
3. **Business Intelligence (Power BI)**: Developed an interactive executive dashboard tracking KPIs (ADR, RevPAR, and Cancellations).
